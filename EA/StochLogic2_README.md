# StochLogic2EA - ロジック② Stochastics マルチタイムフレーム EA

## 概要

Exness XAUUSDm / USDJPY 向けのロジック②を実装したMT5 Expert Advisorです。

**戦略**: 4H/1H/5M足のStochastics(9,3,3)を使ったマルチタイムフレーム戦略

---

## ロジック仕様

### ロングエントリー条件

1. **4H足**: 0〜20圏内でゴールデンクロス（%K が %D を下から上に抜ける）→ ロングモード開始
2. **4H足**: 80〜100圏内でデッドクロスが出るまでロング戦略のみ実行
3. **1H足**: %K > %D（上昇方向）の間のみエントリー可能
4. **5M足**: 0〜20圏内でゴールデンクロス → **ロングエントリー**
5. **イグジット**: 5M足が80〜100圏内でデッドクロス → ポジションクローズ

### ショートエントリー条件

1. **4H足**: 80〜100圏内でデッドクロス → ショートモード開始
2. **4H足**: 0〜20圏内でゴールデンクロスが出るまでショート戦略のみ実行
3. **1H足**: %K < %D（下降方向）の間のみエントリー可能
4. **5M足**: 80〜100圏内でデッドクロス → **ショートエントリー**
5. **イグジット**: 5M足が0〜20圏内でゴールデンクロス → ポジションクローズ

## パラメータ設定

### Stochastics 設定

| パラメータ | デフォルト | 説明 |
|-----------|:---------:|------|
| `InpStochK` | 9 | %K 期間 |
| `InpStochD` | 3 | %D 期間 |
| `InpStochSlowing` | 3 | スローイング |
| `InpOversold` | 20.0 | 売られすぎライン |
| `InpOverbought` | 80.0 | 買われすぎライン |

### ロット管理設定（段階的ロットアップ）

| パラメータ | デフォルト | 説明 |
|-----------|:---------:|------|
| `InpAutoLot` | true | 段階的ロットアップを使用する |
| `InpLotStage1` | 0.01 | フェーズ1ロット（〜¥10万） |
| `InpLotStage2` | 0.05 | フェーズ2ロット（¥10万〜¥50万） |
| `InpLotStage3` | 0.20 | フェーズ3ロット（¥50万〜¥100万） |
| `InpLotStage4` | 0.66 | フェーズ4ロット（¥100万〜） |
| `InpStageThresh1` | 100000 | フェーズ2開始残高（円） |
| `InpStageThresh2` | 500000 | フェーズ3開始残高（円） |
| `InpStageThresh3` | 1000000 | フェーズ4開始残高（円） |

### リスク管理設定

| パラメータ | デフォルト | 説明 |
|-----------|:---------:|------|
| `InpMaxLot` | 1.00 | 最大ロット上限 |
| `InpRuinPct` | 30.0 | 破産ライン（初期残高の%） |
| `InpUseSL` | false | 固定SLを使用する |
| `InpSLPoints` | 500 | 固定SL（ポイント） |
| `InpUseTP` | false | 固定TPを使用する |
| `InpTPPoints` | 1000 | 固定TP（ポイント） |

## 注意事項

1. **破産確率**: ¥50,000スタートで段階的ロットアップを使用した場合、破産確率は約11.5%です
2. **スプレッド**: バックテストは固定スプレッド（GOLD $0.30）で実施。実際のスプレッドは変動します
3. **夏時間**: `InpServerOffset` を夏時間（3）・冬時間（2）で切り替えてください
4. **ブローカー**: Exness 2,000倍レバレッジ推奨（証拠金を最小化）

## インストール方法

1. MT5を起動
2. 「ファイル」→「データフォルダを開く」→「MQL5」→「Experts」
3. `StochLogic2EA.mq5` をコピー
4. MT5で「ツール」→「メタエディター」を開く
5. 左側のナビゲーターから `StochLogic2EA.mq5` を選択
6. 「コンパイル」ボタンをクリック
7. エラーがなければ「ナビゲーター」→「エキスパートアドバイザー」に表示されます
8. チャートにドラッグ&ドロップして起動
