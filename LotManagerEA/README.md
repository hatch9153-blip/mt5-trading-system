# LotManager EA v1.0

MT5用ロット管理・グリッドオーダーEA。GOLD（XAUUSD/XAUUSDmicro）を主対象とした裁量トレードの補助ツールです。

## 主な機能

### 機能1: 平均取得価格の水平線表示
現在チャートの銘柄について、Buy/Sell別の**加重平均取得価格**を自動算出し、チャート上に破線の水平線として表示します。ポジションの追加・決済のたびに自動更新されます。

### 機能2: ポジション数・追加可能数の表示
GUIパネル内に以下の情報をリアルタイム表示します。

| 表示項目 | 内容 |
|----------|------|
| Balance | 口座残高 |
| Max Lots | 残高基準の最大許容ロット数 |
| Current Lots | 現在の保有ロット数 |
| Addable Lots | 追加可能ロット数（ポジション数） |
| Positions | Buy/Sell別のポジション数 |
| Avg Buy / Avg Sell | 平均取得価格 |

**追加可能数の算出基準**: 口座残高1,000円（口座通貨）につき0.1ロット。Micro口座・Standard口座共通。

### 機能3: GUIパネルによる分割指値エントリー
チャート左上に操作パネルを表示し、以下の操作をリアルタイムで行えます。

#### ゾーン設定
- 最大5ゾーンに分割して指値注文を設定可能
- 各ゾーンに「上限価格（High）」「下限価格（Low）」「分割数（Splits）」「TP」「SL」を個別設定

#### SL/TPモード（3種類）
| モード | 説明 |
|--------|------|
| Global | 全注文共通のSL/TPを適用 |
| PerOrder | 各ゾーンのSL/TP欄を個別に使用（注文ごと） |
| PerZone | ゾーン単位でSL/TPを設定 |

#### Auto SL（直近高値・安値ベース）
- `[ON] Auto SL (Swing)` ボタンをONにすると、現在の時間足における直近の高値/安値から$5（`InpSLOffsetDollar`で変更可）離れた位置にSLを自動設定します。
- Buy注文 → 直近安値 - $5
- Sell注文 → 直近高値 + $5

#### 想定損益シミュレーション
- `Calc P/L` ボタンで、設定した全注文がSL/TPに到達した場合の**最大想定損失額**と**最大想定利益額**を表示します。

## インストール方法

1. `LotManager.mq5` を MT5の `MQL5/Experts/` フォルダにコピーします。
2. MetaEditorでコンパイルします（F7キー）。
3. GOLDチャートにEAをドラッグ＆ドロップします。
4. 「自動売買を許可する」にチェックを入れます。

## パラメーター設定

| パラメーター | デフォルト | 説明 |
|-------------|-----------|------|
| InpBaseLot | 0.1 | 1注文あたりの基準ロット数 |
| InpBalancePer1000 | 0.1 | 残高1,000につき許容するロット数 |
| InpSLOffsetDollar | 5 | Auto SL時の高値/安値からのオフセット（ドル） |
| InpAvgBuyColor | DodgerBlue | Buy平均価格線の色 |
| InpAvgSellColor | OrangeRed | Sell平均価格線の色 |
| InpPanelX | 10 | パネルのX座標 |
| InpPanelY | 30 | パネルのY座標 |

## 操作手順（分割指値注文の発注）

1. **ゾーン数を選択**: パネル上部の `1`〜`5` ボタンで使用するゾーン数を選択
2. **価格帯を入力**: 各ゾーンの `High`（上限）と `Low`（下限）を入力
3. **分割数を入力**: `Splits` 欄に等間隔で配置する注文数を入力
4. **SLモードを選択**: `Global` / `PerOrder` / `PerZone` から選択
5. **SL/TPを設定**: 各欄に価格を入力（Auto SLを使う場合はトグルをON）
6. **`Calc P/L` で確認**: 想定損益を確認
7. **`Place Orders` で発注**: 全ゾーンの指値注文を一括発注
8. **`Cancel All` でキャンセル**: このEAが発注した全注文を一括取消

## 注意事項

- 本EAは裁量トレードの補助ツールです。自動でエントリーは行いません。
- `Place Orders` ボタンを押すと即座に指値注文が発注されます。内容を十分確認してから押してください。
- ゾーンのHigh > Low の場合はBuy指値、High < Low の場合はSell指値として処理されます。
- 現在価格より有利な方向の注文はLimit注文、不利な方向はStop注文として自動判定されます。
