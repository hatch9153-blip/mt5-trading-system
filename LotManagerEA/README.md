# LotManager EA v2.0

MT5用ロット管理・グリッドオーダーEA。GOLD（XAUUSD/XAUUSDmicro）を主対象とした裁量トレードの補助ツールです。

## v2.0 新機能

| 機能 | 説明 |
|------|------|
| **動的ゾーンUI** | ゾーン数ボタンを押すと、その数のゾーン入力欄だけが表示される |
| **チャートライン指定** | 各ゾーンのHigh/Low/SL/TPの「L」ボタンを押すと水平線がチャートに出現し、ドラッグ→ダブルクリックで価格を確定 |
| **AutoSL手動変更** | AutoSLのオフセット($)をパネル上のEditフィールドで変更可能 |
| **仮想残高入力** | `VirtBal($)` フィールドに入力すると、その金額でMaxLot計算を行う（入金前の試算用） |
| **ゾーン別MaxRisk/Reward** | 各ゾーンに `MaxRisk($)` / `MaxRwd($)` を設定可能。グローバルのMaxRisk/Rewardも追加 |

## インストール

1. `LotManager.mq5` を MT5 の `MQL5/Experts/` フォルダにコピー
2. MT5のMetaEditorでコンパイル（F7）
3. GOLDチャートにドラッグ＆ドロップ

## GUIパネル操作方法

### ゾーン設定

1. **Zones ボタン（1〜5）** をクリックしてゾーン数を選択 → 選んだ数のゾーン入力欄が表示される
2. 各ゾーンの **High / Low** に価格を入力するか、隣の **「L」ボタン** でチャートライン指定
3. **Splits** に分割数を入力
4. **TP / SL** に価格を入力するか、「L」ボタンでチャートライン指定

### チャートライン指定の手順

1. 「L」ボタンをクリック → チャート上に水平線が表示される
2. 線をドラッグして目的の価格に合わせる
3. **ダブルクリック** で価格を確定（入力欄に自動入力される）
4. ESCキーでキャンセル

### SL/TP設定

- **SL Mode**: `Global`（全ゾーン共通）/ `PerOrder`（注文ごと）/ `PerZone`（ゾーンごと）
- **AutoSL(Swing)**: ONにすると直近の高値/安値からオフセット分離れた位置に自動でSLを設定
  - `Offset($)` フィールドでオフセット金額を変更可能

### 仮想残高（試算用）

- `VirtBal($)` に金額を入力すると、その残高でMaxLot計算を行う
- `0` を入力すると実際の口座残高に戻る
- 残高表示に `[VIRT]` と表示される

### MaxRisk / MaxReward

- **ゾーン別**: 各ゾーンの `MaxRisk($)` / `MaxRwd($)` に上限を設定（0=制限なし）
- **グローバル**: 全ゾーン合計の上限を設定
- `Calc P/L` ボタンで上限適用後の想定損益を確認できる

### 発注・キャンセル

- **Calc P/L**: 現在の設定で想定損益を計算して表示
- **Place Orders**: 設定した全ゾーンの指値注文を一括発注
- **Cancel All**: このEAが発注した全ての指値注文をキャンセル
- **Refresh**: パネルを再描画
- **Read Fields**: 入力欄の値を内部変数に読み込む

## v1.0 との互換性

v1.0のEAとv2.0のEAは異なるMagic Number（v1.0: 202601, v2.0: 202602）を使用するため、同時に稼働しても注文が混在しません。

## パラメーター一覧

| パラメーター | デフォルト | 説明 |
|---|---|---|
| InpBaseLot | 0.1 | 1注文あたりのロット数 |
| InpBalancePer1000 | 0.1 | 残高1000円あたりのロット数 |
| InpSLOffsetDollar | 5 | AutoSLの初期オフセット($) |
| InpAvgBuyColor | DodgerBlue | 平均買値ラインの色 |
| InpAvgSellColor | OrangeRed | 平均売値ラインの色 |
| InpPanelX | 10 | パネルのX座標 |
| InpPanelY | 30 | パネルのY座標 |
