# ロット管理方式 比較バックテスト

ロジック②（4H/1H/5M足 Stochastics(9,3,3)）を用いた3種類のロット管理方式の検証コードです。

## ファイル一覧

| ファイル | 内容 |
|---------|------|
| `backtest_dynamic_lot.py` | 残高比例ロット（¥10,000ごとに0.01lot、毎トレード更新） |
| `backtest_step_lot.py` | 段階的ロット（残高2倍達成ごとにロットを2倍に増やす） |

## 3方式の比較結果サマリー（¥100,000スタート / 2023年1月〜2026年2月）

## ロット管理方式の特徴

### 固定0.01lot
- 最も安全（最大DD: GOLD 3.4% / USDJPY 6.1%）
- 収益額は口座残高に関わらず固定
- 長期運用では資金効率が低下

### 残高比例ロット（¥10,000ごとに0.01lot）
- 最高収益（GOLD +1,257.2% / USDJPY +730.5%）
- 最大DDが高い（GOLD 34.5% / USDJPY 47.4%）
- 残高増加とともに損益額も比例して増大

### 段階的ロット（残高2倍でロットアップ）
- 収益性と安全性のバランスが最も優れる
- ロットアップ回数: GOLD 2回 / USDJPY 3回
- 残高が十分に積み上がってからリスクを増やす設計
- GOLDでは最大DD 22.7%に抑えながら+631.7%を達成

## 推奨運用

- **安全重視**: 固定0.01lot（¥100,000口座）
- **バランス重視**: 段階的ロット（GOLD優先）
- **収益最大化**: 残高比例ロット（上限lot設定を推奨）

## 注意事項

- スプレッド固定: GOLD $0.30 / USDJPY 0.03円
- スリッページ未考慮
- 夏時間（サマータイム）未対応
- 過去の結果は将来の利益を保証するものではありません
